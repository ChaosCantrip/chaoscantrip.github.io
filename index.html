Hi! I'm James

I'm an Electrical Engineering student at Imperial College London. I started programming when I was young, and have started trying to make my way in a career as a Software Engineer.

My Dad has the sole claim to responsibility when it comes to my love of programming. Back when I was 6, he took me to the Big Bang Fair, in Birmingham. At that event, there was a stall for Codecademy teaching basic HTML. That day, I went home, and burned my way through the entire course, learning HTML, CSS and Javascript, and I fell in love with programming forever.

Shortly after that, he got me a Raspberry Pi for Christmas, and a copy of PROGRAMMING RASPBERRY PI: GETTING STARTED WITH PYTHON, by Simon Monk. I learnt to program in Python, and finally got my first taste of real code development.

SharkBot

Python

SharkBot was my first large project.

Utilising Python, and the discord.py library, I built a bot to serve the Discord Server me and my friends use. Originally built to keep track of a #count-to-1000 channel we had going, it fast became my new hobby, adding more features and content as fast as we could come up with it.

Now, it has a plethora of features, from admin features to help us run the server to (free) lootboxes and an ever-expanding item collection game. More info can be found soon at SharkBot Online, though that website is still a work in progress.

Until recently, SharkBot was hosted by myself, running from a Raspberry Pi quite literally scotch taped to my bedroom wall. Unfortunately, I have moved house, and due to network reliability issues, this solution wasn't working as well so he now runs on an AWS EC2 Instance. Changes to the code are uploaded to GitHub, and the bot automatically fetches and applies them when needed.
